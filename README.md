# chat_app

# 파이어베이스 연결
- Authentication: 구글 로그인 가능
- Firestore Database: 실시간 채팅 가능

# 어플 기능
- 메시지 삭제 기능
- 발신 및 수신 시간 표시 기능
- 채팅 상대의 아이디 확인 가능
